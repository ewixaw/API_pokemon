Bienvenue sur Pokédex89

Sur ce site vous pourrez trouver :

  • Un page avec tous les pokémons existants
  • Une page de description par pokémon
  • Différentes fonctionnalités permettant une navigation plus facile entre les pokémons

Pour accéder à ces différentes pages, il vous suffit de passer votre curseur sur le titre "Pokémons" en haut à droite du site pour qu'un menu déroulant apparaisse.

Vous pouvez aussi accéder directement à la fiche d'un pokémon en tapant son nom dans la barre de recherche.
